RCON

What is RCON?

RCON is a protocol that allows server administrators to remotely execute Minecraft commands.

Setting up RCON

  • Go to panel.falixnodes.net and click on your server.
  • Go to the network tab and click on New Allocation.
  • Go to file manager and open server.properties, set `enable-rcon` to true, and restart your server.
  • Go back to network tab and copy the secondary port. (Example)
  • Go back to file manager and open server.properties, set `rcon.port` to the port you copied, and set `rcon.password` to a password you want.
  • Restart your server.

    Connecting to RCON

    Download a RCON client called McRcon

  • Open powershell and do `cd path/to/mcronfolder`
  • do `mcrcon.exe -H serverIPwithoutPort -P secondaryPort -p passwordHere`

    • You should now have access to the console of your minecraft server.
